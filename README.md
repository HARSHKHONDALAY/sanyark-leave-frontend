# Sanyark Leave Frontend

React frontend for the Sanyark Leave Management System technical assignment.

## Stack

- React
- Vite
- React Router
- Axios
- Custom CSS
- Particles.js style space background UI

## Features

- JWT login
- Employee dashboard
- Manager dashboard
- Apply leave
- My leaves
- Manager leave approvals
- Team calendar
- Responsive glassmorphism UI

## Development

```bash
npm install
npm run dev
